
~> To backup pip (Python) packages:
    Step 0> (Restrore Backup)
        Install Python and pip.
        command:
            pip install -r dev/python/requirements.txt
        If you ever use pipx, you can add:
            pipx list --json
        OR 
            run .ps1 files for windows
            run .sh file for Linux/macOS

    Step 1>
        Create a folder named "python" if not exsists already
        command:
            mkdir python
    Step 2>
        To copy pip global packages for Python
        run this command:
            pip freeze > python/requirements.txt
